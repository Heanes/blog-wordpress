== Description ==

This plugin is more powerful than the combination of "simple-lazyload" and "auto-highslide".

This plugin contains four gadgets:

1. max-width limitation for images in the article, configurable, default 600px.
2. true lazyload realization.
3. highslide view effect for images.
4. tracking code setting, no longer worry about theme changes.

本插件是个组合插件，是【simple-lazyload】和【auto-highslide】的升级增强版。

本插件包含四个功能：

1. 文章图片的最大宽度限制，默认600像素
2. 真实的图片迟加载功效
3. 图片弹出放大的浏览效果，使用highslide效果
4. 网站统计代码设置，脱离主题而存在，再也不用操心换主题后统计代码丢失的事了

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `auto-lazyload-and-auto-highslide` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress Background.
